Topic 1: Data Organization Principles

Setting Up Data for Success

Before you can effectively sort and filter data, it's crucial to organize it properly. Well-structured data makes everything else easier and more reliable.

Visual Reference Screenshots:

üì∏ Excel Sorting Guide - Shows proper data organization for sorting üì∏ Google Sheets Filtering - Demonstrates data organization and filtering setup

Essential Data Organization Rules:

  1. Headers in Row 1: Each column should have a clear, descriptive header
  2. One Data Type per Column: Don't mix text and numbers in the same column
  3. No Empty Rows/Columns: Avoid gaps in your data range
  4. Consistent Formatting: Use the same date format, number format, etc.
  5. No Merged Cells: Merged cells break sorting and filtering

Step-by-Step: Preparing Data for Sorting and Filtering

  1. Create proper headers:
    • A1: "Employee Name"
    • B1: "Department"
    • C1: "Hire Date"
    • D1: "Salary"
    • E1: "Performance Rating"
  2. Enter consistent data:
    • Use same date format throughout (MM/DD/YYYY)
    • Format salary as currency
    • Use consistent department names (not "Sales" and "sales")
  3. Check for data quality:
    • No empty rows within the data
    • No extra spaces in text entries
    • Consistent capitalization

Common Data Organization Mistakes:

  • Multiple header rows: Avoid titles above your column headers
  • Inconsistent data entry: "New York" vs "NY" vs "new york"
  • Mixed data types: Numbers stored as text, dates as text
  • Empty rows for spacing: Use formatting instead of empty rows
  • Data in merged cells: Unmerge before sorting/filtering

Topic 2: Basic Sorting

Understanding Sort Operations

Sorting rearranges your data in a specific order (alphabetical, numerical, chronological) while keeping related information in each row together.

Sort Type When to Use Example Result
A to Z (Ascending) Alphabetical order, smallest to largest Names, dates (oldest first) Adams, Brown, Smith
Z to A (Descending) Reverse alphabetical, largest to smallest Salaries (highest first) Smith, Brown, Adams
Custom Sort Specific business logic or multiple criteria Department priority, then by name CEO, Manager, Employee

Step-by-Step: Simple Single-Column Sort

  1. Set up sample data:
    • A1: "Name", B1: "Score", C1: "Grade"
    • A2: "John", B2: 85, C2: "B"
    • A3: "Alice", B3: 92, C3: "A"
    • A4: "Bob", B4: 78, C4: "C"
  2. Select your data:
    • Click anywhere in your data table
    • Excel/Sheets will automatically detect the data range
    • Or manually select A1:C4
  3. Sort by name (A to Z):
    • Excel: Data tab ‚Üí Sort A to Z (with Name column selected)
    • Google Sheets: Data menu ‚Üí Sort range ‚Üí Sort by Column A (A‚ÜíZ)
  4. Verify the result:
    • Names should now be: Alice, Bob, John
    • Notice how the scores and grades stayed with their respective names

Sort Behavior with Different Data Types:

  • Text: Sorts alphabetically (A-Z or Z-A)
  • Numbers: Sorts numerically (smallest to largest or vice versa)
  • Dates: Sorts chronologically (oldest to newest or vice versa)
  • Mixed data: Numbers first, then text, then blanks

Multi-Level Sorting

When you need to sort by multiple criteria, such as department first, then by salary within each department.

Step-by-Step: Multi-Level Sort

  1. Set up employee data:
    • A1: "Name", B1: "Department", C1: "Salary"
    • A2: "John", B2: "Sales", C2: 50000
    • A3: "Alice", B3: "IT", C3: 65000
    • A4: "Bob", B4: "Sales", C4: 55000
    • A5: "Carol", B5: "IT", C5: 60000
  2. Access multi-level sort:
    • Excel: Data tab ‚Üí Sort (the main Sort button, not A-Z)
    • Google Sheets: Data menu ‚Üí Sort range ‚Üí Advanced range sorting options
  3. Set up sort criteria:
    • Primary sort: Department (A to Z)
    • Secondary sort: Salary (largest to smallest)
  4. Execute the sort and verify:
    • IT employees should appear first, with Alice before Carol (higher salary)
    • Then Sales employees, with Bob before John (higher salary)

Topic 3: Filtering Techniques

Understanding Filters

Filtering shows only the rows that meet specific criteria, temporarily hiding the rest. Unlike sorting, filtering doesn't rearrange data‚Äîit just controls what's visible.

AutoFilter

Basic filtering with dropdown arrows on each column header. Great for quick, simple filtering tasks.

Custom Filter

More control over filter criteria, including ranges, multiple conditions, and complex logic.

Text Filters

Filter text columns using contains, starts with, ends with, and other text-specific criteria.

Number Filters

Filter numeric data using greater than, less than, between ranges, and top/bottom values.

Step-by-Step: Applying AutoFilter

  1. Set up employee data:
    • A1: "Name", B1: "Department", C1: "Salary", D1: "Years"
    • Add several rows of employee data
  2. Enable AutoFilter:
    • Excel: Select data ‚Üí Data tab ‚Üí Filter
    • Google Sheets: Select data ‚Üí Data menu ‚Üí Create a filter
  3. Notice the dropdown arrows:
    • Small dropdown arrows appear in each header cell
    • These arrows provide access to filtering options
  4. Apply a simple filter:
    • Click dropdown arrow in Department column
    • Uncheck "Sales" to hide Sales employees
    • Click OK

Filter Indicators:

  • Dropdown arrow changes color: Indicates a filter is active on that column
  • Row numbers have gaps: Shows that some rows are hidden
  • Status bar shows count: Displays how many rows are visible vs. total

Advanced Filtering Options

Step-by-Step: Custom Number Filters

  1. Filter by salary range:
    • Click dropdown in Salary column
    • Choose "Number Filters" ‚Üí "Between"
    • Enter range: 50000 to 70000
    • Only employees with salaries in this range will show
  2. Filter top performers:
    • Click dropdown in Salary column
    • Choose "Number Filters" ‚Üí "Top 10"
    • Adjust to show top 3 or top 25%

Step-by-Step: Text Filters

  1. Filter names starting with specific letter:
    • Click dropdown in Name column
    • Choose "Text Filters" ‚Üí "Begins With"
    • Enter "A" to show only names starting with A
  2. Filter names containing text:
    • Choose "Text Filters" ‚Üí "Contains"
    • Enter "son" to find Johnson, Wilson, etc.

Multiple Filter Criteria

Filter Logic: When you apply filters to multiple columns, Excel shows only rows that meet ALL criteria (AND logic). For example:

  • Department = "IT" AND Salary > 60000
  • Shows only IT employees earning more than $60,000

Step-by-Step: Combining Multiple Filters

  1. Apply first filter:
    • Filter Department to show only "IT" and "Sales"
  2. Apply second filter:
    • Filter Years to show only employees with 5+ years
  3. Apply third filter:
    • Filter Salary to show only salaries above $55,000
  4. Result:
    • Shows only IT or Sales employees with 5+ years earning $55,000+

Topic 4: Data Validation

Controlling Data Entry

Data validation ensures that only specific types of data can be entered into cells, preventing errors and maintaining data quality.

Common Validation Rules:

  • Dropdown Lists: Choose from predefined options
  • Number Ranges: Only allow numbers within specific limits
  • Date Ranges: Restrict dates to specific periods
  • Text Length: Control minimum/maximum character count
  • Custom Formulas: Create complex validation logic

Step-by-Step: Creating Dropdown Lists

  1. Set up your list source:
    • In a separate area (like column F), list valid departments:
    • F1: "Sales", F2: "Marketing", F3: "IT", F4: "HR"
  2. Select target cells:
    • Select the Department column where you want validation
  3. Apply data validation:
    • Excel: Data tab ‚Üí Data Validation
    • Google Sheets: Data menu ‚Üí Data validation
  4. Configure the validation:
    • Allow: List (Excel) or Dropdown (Google Sheets)
    • Source: Select your F1:F4 range
    • Check "In-cell dropdown" if available
  5. Test the validation:
    • Click in a Department cell
    • Notice the dropdown arrow appears
    • Try entering invalid data‚Äîit should be rejected

Step-by-Step: Number Range Validation

  1. Select salary column cells
  2. Open Data Validation dialog
  3. Configure number validation:
    • Allow: Decimal (Excel) or Number (Google Sheets)
    • Data: Between
    • Minimum: 30000
    • Maximum: 150000
  4. Add helpful messages:
    • Input Message: "Enter salary between $30,000 and $150,000"
    • Error Message: "Salary must be between $30,000 and $150,000"
  5. Test the validation:
    • Try entering 25000 (should be rejected)
    • Try entering 175000 (should be rejected)
    • Enter 75000 (should be accepted)

Step-by-Step: Date Range Validation

  1. Select hire date column
  2. Configure date validation:
    • Allow: Date
    • Data: Between
    • Start date: 1/1/2000 (no hiring before this date)
    • End date: TODAY() (no future hire dates)
  3. Test with various dates

Feature Microsoft Excel Google Sheets
