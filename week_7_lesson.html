Topic 1: Understanding Boolean Logic

TRUE and FALSE - The Foundation of Logic

Before diving into logical functions, it's essential to understand Boolean logic. Every logical expression in spreadsheets evaluates to either TRUE or FALSE.

Visual Reference Screenshots:

📸 Excel IF Function Guide - Basic IF statement syntax and examples 📸 Google Sheets IF Function - Implementation and practical uses

Boolean Values in Practice:

  • TRUE: The condition is met (1 in numeric context)
  • FALSE: The condition is not met (0 in numeric context)
  • Comparisons: =A1>10 returns TRUE if A1 is greater than 10
  • Text comparisons: =A1="Apple" returns TRUE if A1 contains "Apple"

Step-by-Step: Basic Boolean Expressions

  1. Set up test data:
    • A1: 15, A2: 8, A3: "Apple", A4: "Orange"
  2. Create simple comparisons:
    • B1: =A1>10 (returns TRUE)
    • B2: =A2>10 (returns FALSE)
    • B3: =A3="Apple" (returns TRUE)
    • B4: =A4="Apple" (returns FALSE)
  3. Test different operators:
    • C1: =A1>=15 (greater than or equal)
    • C2: =A2<>8 (not equal - FALSE because A2 equals 8)
    • C3: =A3<>A4 (TRUE because "Apple" doesn't equal "Orange")
Operator Meaning Example Result
= Equal to =A1=10 TRUE if A1 equals 10
<> Not equal to =A1<>10 TRUE if A1 doesn't equal 10
> Greater than =A1>10 TRUE if A1 is greater than 10
>= Greater than or equal =A1>=10 TRUE if A1 is 10 or greater
< Less than =A1<10 TRUE if A1 is less than 10
<= Less than or equal =A1<=10 TRUE if A1 is 10 or less

Topic 2: IF Functions - Basic Decision Making

The IF Function Syntax

The IF function is the cornerstone of logical operations in spreadsheets. It allows you to perform different actions based on whether a condition is true or false.

Basic IF Syntax:
=IF(logical_test, value_if_true, value_if_false)

Step-by-Step: Creating Your First IF Statement

  1. Set up test scenario:
    • A1: "Score", B1: "Grade"
    • A2: 85
  2. Create basic IF statement:
    • B2: =IF(A2>=60,"Pass","Fail")
    • This checks if A2 is 60 or greater
    • If TRUE, displays "Pass"
    • If FALSE, displays "Fail"
  3. Test with different values:
    • Change A2 to 45 - should show "Fail"
    • Change A2 to 75 - should show "Pass"
    • Change A2 to exactly 60 - should show "Pass"

IF Statements with Different Data Types

Numeric IF Statements

=IF(A1>1000,"High","Low")
=IF(B1<=0,0,B1*0.1)

Use for salary grades, inventory levels, performance metrics

Text IF Statements

=IF(A1="Manager","Boss","Employee")
=IF(B1="","Missing Data",B1)

Use for department codes, status indicators, data validation

Date IF Statements

=IF(A1>TODAY(),"Future","Past")
=IF(B1

Use for deadline tracking, age calculations, time-based analysis

Step-by-Step: Employee Bonus Calculator

  1. Set up employee data:
    • A1: "Name", B1: "Sales", C1: "Bonus"
    • A2: "John", B2: 15000
    • A3: "Alice", B3: 22000
    • A4: "Bob", B4: 8000
  2. Create bonus formula:
    • C2: =IF(B2>=20000,B2*0.1,B2*0.05)

IF Function Best Practices:

  • Clear conditions: Make your logical tests easy to understand
  • Consistent data types: Ensure value_if_true and value_if_false are compatible
  • Handle edge cases: Consider what happens with blank cells or errors
  • Use quotes for text: Always quote literal text values

Topic 3: Logical Operators (AND, OR, NOT)

Combining Multiple Conditions

Real-world decisions often depend on multiple factors. Logical operators allow you to combine several conditions in a single formula.

AND All conditions must be true =AND(condition1, condition2, ...) ALL conditions are TRUE
OR At least one condition must be true =OR(condition1, condition2, ...) ANY condition is TRUE
NOT Reverses true/false =NOT(condition) The condition is FALSE

Step-by-Step: Using AND Function

  1. Set up employee qualification data:
    • A1: "Name", B1: "Experience", C1: "Education", D1: "Qualified"
    • A2: "John", B2: 3, C2: "Bachelor", D2: (formula here)

Step-by-Step: Using OR Function

  1. Set up discount eligibility:

    • A2: "Sarah", B2: 67, C2: 2, D2: 150
    • A3: "Mike", B3: 45, C3: 8, D3: 75

  2. Create discount rules:
    • Discount if: Senior (65+) OR Long-term member (5+ years) OR Large purchase ($250+)

Step-by-Step: Using NOT Function

  1. Set up inventory status:
    • A1: "Product", B1: "In Stock", C1: "Status"
    • A2: "Laptop", B2: TRUE
    • A3: "Mouse", B3: FALSE
  2. Create status using NOT:
    • C2: =IF(NOT(B2),"Order Now","Available")
    • This shows "Order Now" when NOT in stock (when B2 is FALSE)
  3. Results:
    • Laptop: Available (NOT(TRUE) = FALSE, so else condition)
    • Mouse: Order Now (NOT(FALSE) = TRUE, so if condition)

Topic 4: Nested IF Statements

Building Complex Decision Trees

When you need to evaluate multiple conditions in sequence, nested IF statements allow you to create sophisticated decision-making formulas.

Nested IF Syntax:
=IF(condition1, result1, IF(condition2, result2, IF(condition3, result3, final_result)))

Nested IF Limitations:

  • Excel: Maximum of 64 nested IF statements
  • Google Sheets: Practically unlimited, but performance may degrade
  • Readability: More than 3-4 levels become hard to understand
  • Alternative: Consider using lookup tables or IFS function for complex scenarios

Step-by-Step: Letter Grade Calculator

  1. Set up grade data:
    • A1: "Student", B1: "Score", C1: "Letter Grade"
    • A2: "John", B2: 87
    • A3: "Alice", B3: 94
    • A4: "Bob", B4: 73
  2. Create nested IF for letter grades:
    • C2: =IF(B2>=90,"A",IF(B2>=80,"B",IF(B2>=70,"C",IF(B2>=60,"D","F"))))
  3. Break down the logic:
    • If score ≥ 90: "A"
    • Else if score ≥ 80: "B"
    • Else if score ≥ 70: "C"
    • Else if score ≥ 60: "D"
    • Else: "F"
  4. Copy formula and verify:
    • John (87): "B"
    • Alice (94): "A"
    • Bob (73): "C"

Step-by-Step: Employee Performance Rating

  1. Set up performance data:
    • A1: "Employee", B1: "Sales %", C1: "Customer Rating", D1: "Performance"
    • A2: "John", B2: 110, C2: 4.5
    • A3: "Alice", B3: 95, C3: 4.8
    • A4: "Bob", B4: 85, C4: 3.2
  2. Create complex rating system:
    • D2: =IF(AND(B2>=100,C2>=4.5),"Excellent",IF(AND(B2>=90,C2>=4.0),"Good",IF(AND(B2>=80,C2>=3.5),"Satisfactory","Needs Improvement")))
  3. Understand the nested logic:
    • Excellent: Sales ≥100% AND Rating ≥4.5
    • Good: Sales ≥90% AND Rating ≥4.0
    • Satisfactory: Sales ≥80% AND Rating ≥3.5
    • Otherwise: Needs Improvement

Alternative to Complex Nested IFs: The IFS Function

IFS Function (Excel 2016+, Google Sheets): A cleaner alternative to nested IFs

=IFS(condition1, result1, condition2, result2, condition3, result3, ...)

Example: =IFS(B2>=90,"A",B2>=80,"B",B2>=70,"C",B2>=60,"D",TRUE,"F")

Topic 5: Error Handling with IFERROR

Graceful Error Management

Errors are inevitable in complex formulas. IFERROR allows you to handle errors gracefully by providing alternative values or actions when formulas fail.

Common Spreadsheet Errors:

  • #DIV/0!: Division by zero
  • #VALUE!: Wrong data type (text where number expected)
  • #REF!: Invalid cell reference
  • #N/A: Value not available (common in lookups)
  • #NAME?: Formula name not recognized
IFERROR Syntax:
=IFERROR(formula, value_if_error)

Step-by-Step: Safe Division Formula

  1. Set up division scenario:
    • A1: "Total Sales", B1: "Days", C1: "Daily Average"
    • A2: 15000, B2: 30
    • A3: 8000, B3: 0 (will cause error)
    • A4: 12000, B4: 24
  2. Create error-prone formula:
    • C2: =A2/B2 (works fine: 500)
    • C3: =A3/B3 (shows #DIV/0! error)
  3. Add error handling:
    • C2: =IFERROR(A2/B2,"No data")
    • C3: =IFERROR(A3/B3,"No data")
    • Now C3 shows "No data" instead of error

Step-by-Step: Percentage Calculation with Error Handling

  1. Set up percentage scenario:
    • A1: "Completed", B1: "Total", C1: "Percentage"
    • A2: 45, B2: 60
    • A3: 30, B3: 0
    • A4: "", B4: 40 (empty cell)
  2. Create robust percentage formula:
    • C2: =IFERROR(A2/B2*100&"%","Invalid data")
    • This handles division by zero and empty cells
  3. Results:
    • C2: "75%" (45/60*100)
    • C3: "Invalid data" (division by zero)
    • C4: "Invalid data" (empty cell in numerator)

Combining IFERROR with Logical Functions

Advanced Error Handling Example

  1. Set up grade calculation with potential errors:
    • A1: "Student", B1: "Points", C1: "Max Points", D1: "Grade"
    • Include scenarios: normal data, zeros, blanks, text
  2. Create comprehensive formula:
    • =IFERROR(IF(AND(B2<>"",C2<>"",C2>0),B2/C2*100,"Missing Data"),"Invalid Data")
  3. This formula handles:
    • Empty cells
    • Division by zero
    • Invalid data types
    • Provides meaningful error messages

Error Handling Best Practices:

  • Anticipate errors: Think about what could go wrong with your formulas
  • Meaningful messages: Use descriptive error messages, not just "Error"
  • Data validation: Prevent errors at input time when possible
  • Test edge cases: Try your formulas with zeros, blanks, and extreme values
