Topic 1: Introduction to Data Analysis

What is Data Analysis?

Data analysis involves examining, cleaning, transforming, and modeling data to discover useful information, inform conclusions, and support decision-making. In spreadsheets, we have powerful tools to perform various types of analysis.

Visual Reference Screenshots:

Types of Data Analysis:

  • Descriptive Analysis: What happened? (averages, totals, trends)
  • Diagnostic Analysis: Why did it happen? (correlations, patterns)
  • Predictive Analysis: What might happen? (forecasting, trends)
  • Prescriptive Analysis: What should we do? (recommendations, optimization)

Summarization

Grouping and aggregating data to see totals, averages, counts, and other summary statistics.

  • Sales by region
  • Average performance by department
  • Product category totals

Comparison

Comparing different groups, time periods, or categories to identify differences and similarities.

  • Year-over-year growth
  • Department performance
  • Product comparisons

Trend Analysis

Examining data over time to identify patterns, cycles, and long-term trends.

  • Monthly sales trends
  • Seasonal patterns
  • Growth trajectories

What-if Analysis

Testing different scenarios to understand potential outcomes and make informed decisions.

  • Budget scenarios
  • Price sensitivity
  • Resource allocation

Topic 2: Pivot Tables - Data Summarization

Understanding Pivot Tables

Pivot tables are one of the most powerful tools for data analysis. They allow you to quickly summarize, analyze, explore, and present large amounts of data by reorganizing and aggregating it in meaningful ways.

Pivot Table Components:
‚Ä¢ Rows: Categories to group data vertically
‚Ä¢ Columns: Categories to group data horizontally
‚Ä¢ Values: Data to be summarized (sum, count, average, etc.)
‚Ä¢ Filters: Criteria to limit which data is included

Step-by-Step: Creating Your First Pivot Table

  1. Set up sales data:
    • A1: "Date", B1: "Salesperson", C1: "Product", D1: "Category", E1: "Amount"
    • Add 20+ rows of sample sales data with various dates, people, products, and amounts
    • Include multiple categories like Electronics, Clothing, Books
    • Use dates spanning several months
  2. Create the pivot table:
    • Excel: Select data ‚Üí Insert tab ‚Üí PivotTable
    • Google Sheets: Select data ‚Üí Insert menu ‚Üí Pivot table
    • Choose to place in new worksheet
  3. Configure basic layout:
    • Rows: Drag "Category" to Rows area
    • Values: Drag "Amount" to Values area (should auto-sum)
    • You now have total sales by category
  4. Add more detail:
    • Rows: Add "Product" below "Category"
    • Now you see category totals with product breakdowns

Advanced Pivot Table Techniques

Step-by-Step: Multi-Dimensional Analysis

  1. Create cross-tabulation:
    • Rows: Salesperson
    • Columns: Category
    • Values: Amount (Sum)
    • This shows sales by person and category in a matrix format
  2. Add time analysis:
    • Rows: Date (grouped by month)
    • Columns: Category
    • Values: Amount (Sum)
    • Shows monthly sales trends by category
  3. Group dates automatically:
    • Right-click on date field in pivot table
    • Choose "Group" and select Months, Quarters, or Years

Step-by-Step: Different Value Calculations

  1. Change summary function:
    • Right-click on "Sum of Amount" in Values area
    • Choose "Value Field Settings" or "Summarize by"
    • Try Average, Count, Maximum, Minimum
  2. Add calculated fields:
    • Excel: PivotTable Tools ‚Üí Fields, Items & Sets ‚Üí Calculated Field
    • Google Sheets: Add formula in Values area
    • Example: Create "Average Sale Size" = Amount / Count
  3. Show percentages:
    • Right-click value field ‚Üí "Show Values As"
    • Choose "% of Grand Total" or "% of Row Total"

Pivot Table Best Practices:

  • Clean data first: Remove duplicates, fix inconsistencies

  • Name your fields: Use clear, descriptive field names

Topic 3: Statistical Functions

Essential Statistical Functions

Statistical functions help you understand your data through measures of central tendency, variation, and relationships between variables.

Function Purpose Formula Use Case
AVERAGE Mean value =AVERAGE(range) Typical performance, average sales
MEDIAN Middle value =MEDIAN(range) Central tendency when outliers present
MODE Most frequent value =MODE(range) Most common score, popular item
STDEV Standard deviation =STDEV(range) Measure of variability, consistency
CORREL Correlation coefficient =CORREL(array1,array2) Relationship strength between variables
PERCENTILE Value at specific percentile =PERCENTILE(range,k) Performance benchmarks, rankings

Step-by-Step: Descriptive Statistics Analysis

  1. Set up test score data:
    • A1: "Student", B1: "Math Score", C1: "Science Score"
    • Add 20 students with scores ranging from 60-100
  2. Calculate basic statistics:
    • E1: "Statistic", F1: "Math", G1: "Science"
    • E2: "Average", F2: =AVERAGE(B2:B21), G2: =AVERAGE(C2:C21)
    • E3: "Median", F3: =MEDIAN(B2:B21), G3: =MEDIAN(C2:C21)
    • E4: "Mode", F4: =MODE(B2:B21), G4: =MODE(C2:C21)
    • E5: "Std Dev", F5: =STDEV(B2:B21), G5: =STDEV(C2:C21)
  3. Add percentile analysis:
    • E6: "90th %ile", F6: =PERCENTILE(B2:B21,0.9)
    • E7: "75th %ile", F7: =PERCENTILE(B2:B21,0.75)
    • E8: "25th %ile", F8: =PERCENTILE(B2:B21,0.25)
  4. Calculate correlation:
    • E9: "Correlation", F9: =CORREL(B2:B21,C2:C21)
    • This shows how Math and Science scores relate

Advanced Statistical Analysis

Step-by-Step: Performance Distribution Analysis

  1. Create performance categories:
    • Use statistical measures to define performance bands
    • High: Above (Average + 1 Std Dev)
    • Medium: Between (Average ¬± 1 Std Dev)
    • Low: Below (Average - 1 Std Dev)
  2. Calculate category boundaries:
    • High Cutoff: =AVERAGE(B2:B21)+STDEV(B2:B21)
    • Low Cutoff: =AVERAGE(B2:B21)-STDEV(B2:B21)
  3. Categorize each student:
    • D2: =IF(B2>$F$11,"High",IF(B2<$F$12,"Low","Medium"))
    • Copy down for all students
  4. Count students in each category:
    • =COUNTIF(D2:D21,"High")
    • =COUNTIF(D2:D21,"Medium")
    • =COUNTIF(D2:D21,"Low")

Step-by-Step: Trend Analysis with Statistics

  1. Set up time series data:
    • Monthly sales data for 12 months
    • A1: "Month", B1: "Sales"
  2. Calculate trend statistics:
    • Growth rate: =(B13-B2)/B2 (annual growth)
    • Average monthly change: =AVERAGE((B3:B13-B2:B12)/B2:B12)
    • Volatility: =STDEV(B2:B13)/AVERAGE(B2:B13)
  3. Identify outliers:
    • Calculate z-scores: =(B2-$F$15)/$F$16
    • Flag outliers: =IF(ABS(C2)>2,"Outlier","")

Topic 4: What-if Analysis

Introduction to What-if Analysis

What-if analysis allows you to test different scenarios by changing input values and observing the impact on results. This is crucial for planning, budgeting, and decision-making.

Types of What-if Analysis:

  • Goal Seek: Find the input value needed to achieve a specific goal
  • Data Tables: See how changing one or two variables affects results
  • Scenario Manager: Compare multiple complete scenarios
  • Sensitivity Analysis: Understand which variables have the biggest impact

Step-by-Step: Goal Seek Analysis

  1. Set up loan payment model:
    • A1: "Loan Amount", B1: 250000
    • A2: "Interest Rate", B2: 0.05 (5%)
    • A3: "Term (Years)", B3: 30
    • A4: "Monthly Payment", B4: =PMT(B2/12,B3*12,-B1)
  2. Use Goal Seek to find affordable loan:
    • Excel: Data tab ‚Üí What-If Analysis ‚Üí Goal Seek
    • Google Sheets: Extensions ‚Üí Add-ons ‚Üí Get add-ons ‚Üí Search "Goal Seek"
    • Set cell: B4 (monthly payment)
    • To value: 1200 (target payment)
    • By changing cell: B1 (loan amount)
  3. Interpret results:
    • Goal Seek finds the maximum loan amount for $1,200/month payment
    • Result shows affordable loan amount is approximately $223,000

Step-by-Step: Data Table Analysis

  1. Create sales projection model:
    • A1: "Price", B1: 100
    • A2: "Units Sold", B2: 1000
    • A3: "Variable Cost", B3: 60
    • A4: "Fixed Costs", B4: 25000
    • A5: "Profit", B5: =B1*B2-B3*B2-B4
  2. Set up one-variable data table:
    • D1: "Price Scenarios"
    • D2: 80, D3: 90, D4: 100, D5: 110, D6: 120
    • E1: =B5 (reference to profit formula)
  3. Create data table:
    • Select range D1:E6
    • Excel: Data tab ‚Üí What-If Analysis ‚Üí Data Table
    • Column input cell: B1 (price)
    • Google Sheets: Use array formula or manual calculations
  4. Analyze results:
    • See how profit changes with different prices
    • Identify optimal pricing strategy

Step-by-Step: Two-Variable Data Table

  1. Extend the analysis with two variables:
    • Test different combinations of price and units sold
  2. Set up two-variable table:
    • G1: =B5 (profit formula)
    • G2: 80, G3: 90, G4: 100, G5: 110, G6: 120 (prices)
    • H1: 800, I1: 1000, J1: 1200, K1: 1400 (units)
  3. Create two-variable data table:
    • Select range G1:K6
    • Excel: Data Table with row input (units) and column input (price)
    • Results show profit for each price/volume combination
