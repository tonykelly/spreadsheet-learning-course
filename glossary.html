Spreadsheet Glossary

Essential Terms for Excel & Google Sheets Mastery

How to Use This Glossary

This comprehensive glossary contains all essential spreadsheet terminology from the course, organized alphabetically. Use the letter navigation below to quickly jump to terms starting with specific letters, or use Ctrl+F (Cmd+F on Mac) to search within the page.

Navigation Tips:

  • Click letter links to jump to terms starting with that letter
  • Terms include context from lessons and links to related concepts
  • Use browser search (Ctrl+F) to find specific terms quickly
Quick Navigation: A C D F G H M N O P R S T W

A

Absolute Reference
A cell reference that never changes when copied, indicated by dollar signs ($A$1).
Use for constants like tax rates, lookup tables, or any value that should remain fixed when formulas are copied.
Related: Relative Reference
Active Cell
The currently selected cell in a spreadsheet, highlighted with a colored border. Only one cell can be active at a time.
When you click on a cell or navigate with arrow keys, it becomes the active cell where you can enter data.
Related: Cell
Arguments
The data that functions need to work with, enclosed in parentheses and separated by commas.
For example, in =SUM(A1:A5), the range A1:A5 is the argument telling SUM which cells to add.
Related: Function
Ascending Sort
Sorting data from smallest to largest, A to Z, or oldest to newest, depending on the data type.
Most common sort order for organizing data in logical, progressive order.
Related: Descending Sort
Auto-Fill
A feature that automatically continues patterns or copies data when you drag the fill handle to adjacent cells.
Can extend number sequences (1, 2, 3...), dates (Jan, Feb, Mar...), or copy formulas with adjusted references.
Related: Fill Handle
AutoFilter
Basic filtering feature that adds dropdown arrows to column headers for quick data filtering.
Enables users to show/hide rows based on column values without complex setup.
Related: Filtering
AVERAGE Function
A statistical function that calculates the arithmetic mean of a group of numbers.
Syntax: =AVERAGE(range). Automatically ignores text values and empty cells in the calculation.
Related: SUM Function

C

Cell
An individual rectangle in the spreadsheet grid where data is entered. The intersection of a row and column.
Cells are the fundamental building blocks of spreadsheets, holding text, numbers, formulas, or dates.
Related: Cell Address
Cell Address
A unique identifier for each cell, combining its column letter and row number (e.g., A1, B5, Z100).
Cell addresses are used in formulas to reference data in other cells.
Related: Name Box
Cell Borders
Lines that can be added around or inside cells to improve visual organization and readability.
Common border styles include all borders, outline borders, and specific edge borders (top, bottom, left, right).
Related: Fill Color
Cell Reference
An address used in formulas to point to other cells, enabling dynamic calculations that update when referenced data changes.
Can be relative (A1), absolute ($A$1), or mixed ($A1 or A$1) depending on copying behavior needed.
Related: Formula
COUNT Function
A statistical function that counts how many cells in a range contain numbers.
Only counts numeric values; use COUNTA to count all non-empty cells including text.
Related: COUNTA Function
Currency Formatting
Number formatting that displays monetary values with currency symbols (like $) and appropriate decimal places.
Automatically aligns numbers to the right and can handle negative values in various formats.
Related: Number Formatting

D

Data Types
Categories of information that can be entered in spreadsheet cells: text, numbers, and dates.
Understanding data types is crucial because they determine how data behaves in calculations and formatting.
Related: Text, Numbers, Dates
Data Validation
Rules that control what type of data can be entered into specific cells, preventing errors and ensuring data quality.
Can create dropdown lists, restrict number ranges, validate dates, or control text length.
Related: Dropdown List
Date Formatting
Various ways to display dates, such as 1/15/2024, January 15, 2024, or Jan-15-24.
Excel recognizes dates automatically and can convert between different display formats while maintaining the underlying date value.
Related: Date Functions
Date Functions
Functions that work with dates and times, such as TODAY(), NOW(), YEAR(), MONTH(), and DAY().
Essential for date calculations, age calculations, and time-based analysis.
Related: TODAY Function
Descending Sort
Sorting data from largest to smallest, Z to A, or newest to oldest, depending on the data type.
Useful for finding top performers, highest values, or most recent entries.
Related: Ascending Sort
Dollar Sign ($)
Symbol used to "lock" parts of cell references, preventing them from changing when formulas are copied.
$A locks the column, $1 locks the row, $A$1 locks both. Press F4 in Excel to cycle through reference types.
Related: Absolute Reference
Dropdown List
A data validation feature that allows users to select from a predefined list of options.
Ensures consistent data entry and reduces typos by limiting input to valid choices.
Related: Data Validation

F

Fill Color
The background color of cells, used to highlight important data or organize information visually.
Commonly used for headers (light blue/gray), important data (light yellow), and alternating rows (very light gray).
Related: Cell Borders
Fill Handle
A small square in the bottom-right corner of the active cell used to copy formulas or extend patterns.
Click and drag the fill handle to auto-fill data series or copy formulas to adjacent cells.
Related: Auto-Fill
Filtering
Displaying only rows that meet specific criteria while temporarily hiding the rest.
Unlike sorting, filtering doesn't rearrange data—it controls visibility to focus on relevant information.
Related: Sorting
Format Painter
A tool that copies all formatting from one cell and applies it to other cells quickly.
Represented by a paintbrush icon; double-click to apply formatting to multiple selections.
Related: Cell Formatting
Formula
Any calculation instruction that starts with an equals sign (=), including both simple math and complex functions.
Examples include =A1+B1 (simple formula) and =SUM(A1:A10) (function-based formula).
Related: Function
Formula Bar
The area that shows the actual content of the active cell, including formulas, while the cell displays the calculated result.
Essential for editing formulas and seeing the underlying calculation behind a cell's displayed value.
Related: Formula
Freeze Panes
A feature that keeps selected rows or columns visible while scrolling through large datasets.
Useful for keeping headers visible when working with extensive data tables.
Related: Headers
Function
A pre-built formula with a specific name that performs a particular calculation or operation.
Functions have names (like SUM, AVERAGE) and require arguments in parentheses to specify what data to work with.
Related: Formula

G

Grid System
The structure of rows and columns that forms the foundation of a spreadsheet layout.
Understanding the grid system is essential for navigation and data organization.
Related: Rows and Columns

H

Headers
Descriptive labels in row 1 that identify what data each column contains.
Essential for proper data organization and required for effective sorting and filtering operations.
Related: Data Organization

M

MAX Function
A function that finds and returns the largest value in a range of cells.
Syntax: =MAX(range). Useful for finding highest scores, maximum values, or latest dates.
Related: MIN Function
MIN Function
A function that finds and returns the smallest value in a range of cells.
Syntax: =MIN(range). Useful for finding lowest scores, minimum values, or earliest dates.
Related: MAX Function
Mixed Reference
A cell reference where either the column or row is fixed ($A1 or A$1), but not both.
$A1 keeps column A fixed while row changes; A$1 keeps row 1 fixed while column changes. Useful for table calculations.
Related: Absolute Reference
Multi-Level Sort
Sorting by multiple criteria simultaneously, such as department first, then by salary within each department.
Allows for complex data organization with primary and secondary sort criteria.
Related: Sorting

N

Name Box
A box that displays the address of the active cell. Located in the upper-left corner in Excel, and on the left side of the formula bar in Google Sheets.
Also used to navigate directly to specific cells by typing their address or named range.
Related: Named Range
Named Range
A meaningful name assigned to a cell or range of cells, making formulas easier to read and maintain.
Instead of =B10*$C$1, you can write =SalesTotal*TaxRate. Makes formulas self-documenting and easier to audit.
Related: Cell Reference
Numbers
Numeric data including integers, decimals, and percentages that automatically align to the right in cells.
Used for calculations, measurements, and quantitative analysis. Can be formatted as currency, percentages, or with specific decimal places.
Related: Data Types

O

Operators
Mathematical symbols used in formulas: + (addition), - (subtraction), * (multiplication), / (division), ^ (exponentiation).
Follow order of operations (PEMDAS): Parentheses, Exponents, Multiplication/Division, Addition/Subtraction.
Related: Order of Operations
Order of Operations
Mathematical rules that determine the sequence in which operations are performed in formulas (PEMDAS/BODMAS).
Excel follows standard math rules: Parentheses first, then Exponents, then Multiplication/Division (left to right), then Addition/Subtraction (left to right).
Related: Operators

P

Percentage Formatting
Number formatting that displays decimal values as percentages (0.25 becomes 25%).
Excel automatically multiplies the decimal by 100 and adds the % symbol.
Related: Currency Formatting

R

Range
A group of cells, specified using start and end addresses separated by a colon (A1:A10, B2:D5).
Used in functions like SUM, AVERAGE, and COUNT to work with multiple cells at once.
Related: Range Notation
Range Notation
The syntax for specifying multiple cells using a colon (:) between start and end addresses.
Examples: A1:A10 (column), A1:E1 (row), A1:C5 (rectangle), A:A (entire column), 1:1 (entire row).
Related: Range
Relative Reference
A cell reference that changes when copied to other cells (A1 becomes B1 when copied right, A2 when copied down).
Default reference type; enables formulas to adjust automatically when applied to different rows or columns.
Related: Absolute Reference
ROUND Function
A function that rounds numbers to a specified number of decimal places.
Syntax: =ROUND(number, digits). Use positive digits for decimal places, negative for tens/hundreds.
Related: Number Formatting

S

Sheet Tabs
Tabs at the bottom of the spreadsheet window used to navigate between different worksheets in a workbook.
Allow organization of related data across multiple sheets within a single file.
Related: Worksheet
Sorting
Rearranging data in a specific order (alphabetical, numerical, or chronological) while keeping related information in each row together.
Essential for data analysis, finding patterns, and organizing information for better understanding.
Related: Filtering
SUM Function
The most commonly used function that adds up all numbers in a specified range or list of cells.
Syntax: =SUM(range). Can accept multiple ranges: =SUM(A1:A5,C1:C5). AutoSum button (Σ) automatically creates SUM formulas.
Related: AVERAGE Function

T

Text
Non-numeric data including words, descriptions, and labels that automatically align to the left in cells.
Used for headers, names, descriptions, and any information that won't be used in calculations.
Related: Data Types
Text Functions
Functions that manipulate text data, including CONCATENATE, LEFT, RIGHT, MID, and LEN.
Used for combining text, extracting parts of text strings, and analyzing text length.
Related: CONCATENATE Function
3D Reference
A reference that spans across multiple worksheets, allowing calculations across sheets.
Syntax: Sheet1:Sheet3!A1 references cell A1 across sheets 1, 2, and 3. Useful for consolidating data from multiple sheets.
Related: Worksheet Reference
TODAY Function
A date function that returns the current date, updating automatically each day.
Syntax: =TODAY() (no arguments needed). Useful for calculating ages, days elapsed, or date-based conditions.
Related: Date Functions

W

Workbook
The entire spreadsheet file containing one or more worksheets. Called "Spreadsheet" in Google Sheets.
Like a complete notebook that can contain multiple pages (worksheets) of related data.
Related: Worksheet
Worksheet
A single page or tab within a workbook containing the grid of cells where data is entered and manipulated.
Each worksheet can contain different types of data and calculations while remaining part of the same file.
Related: Workbook
